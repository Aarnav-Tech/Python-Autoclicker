🐍 Python Autoclicker

Fast, simple, and customizable Python-based autoclicker for Windows.

🖱️ Download .exe (v1) 📦 Source Code (.zip)
⚠️ This autoclicker uses the msvcrt module which only works on Windows. You may encounter errors on macOS/Linux.

🛠️ Features

  • ✔ Ultra-lightweight and fast
  • ✔ Set custom click intervals and button
  • ✔ Global hotkeys to start and stop
  • ✔ Only one dependency: pynput

📦 How to Use

  1. Download the EXE or the source code above.
  2. EXE: Just run it on Windows. No setup needed.
  3. Source: Make sure Python 3.10+ is installed.
  4. Install the dependency: pip install pynput
  5. Run it via terminal: python autoclicker.py
