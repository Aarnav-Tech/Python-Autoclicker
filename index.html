🐍 Python Autoclicker

Fast, simple, and customizable autoclicker written in Python.

⬇ Download ZIP ⭐ View on GitHub

🛠️ Features

  • ✔ Cross-platform (Windows, Linux, macOS)
  • ✔ Choose click interval and button
  • ✔ Start/stop with hotkeys
  • ✔ No installation required

📦 How to Use

  1. Download and unzip the repo.
  2. Install Python 3.10+ if not already installed.
  3. Install dependencies: pip install pynput
  4. Run: python autoclicker.py
